Pre-Registration and Billing System

About the System

The system is for Tubod-Baroy Water Disctrict in Pre-Registration, Satellite Collection and Billing System which aims to convert the manual process of the company into an online that will help solve and minimize its problems and offer its clients greater quality of service. The proposed system will enrich the manual process that the company uses.

The structure of the system is in a web-based platform. It is designed using PHP scripting language which the program is created and uses MySQL for the database. The system uses x10hosting.com as its web server.

Objectives of the Study

This study aims to design and develop a system for Tubod-Baroy Water District. The following are the foreseen solutions to the problems that are specified in the statement of the problem.

Specifically, this study aims:

  • Design and create a database system that stores the result of client’s monthly bill.
  • Construct a program that automatically compute the monthly bill based on the meter reading.
  • To create an organized database and provide safer storage for all files.
  • To build a system that will minimize the processing period of transactions between the employees and the clients.
  • To establish a satellite collection system that accepts payment.

Note: This system is created during my thesis where my main role is a developer and a writer in the introductory documentation.

GIS Store Locator

About the Site

A GIS (Geographic Information System) is a site designed to capture, store, manipulate, analyze, manage and present spatial or geographic data. This system uses API from Google to locate all 7-Eleven Stores in the entire Philippines. It is entirely clear that 7-eleven is known nationwide and has expanded all over the entire country. However, The system is controlled to display 10 markers only in each island group of the Philippines which is the Luzon, Visayas and Mindanao.

Currently, the system will locate the 7-Eleven stores through a manual data input in the form by a 7-Eleven employee.

The system needs:

  • Name of the 7-Eleven branch store
  • Address
  • Latitude
  • Longitude

Note: This site is created for reference purposes only.
